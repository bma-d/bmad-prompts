# Procedure
1. Launch subagents per epic to inspect story-level AC/schema quality.
2. Consolidate findings by severity and scope.
3. Patch story docs for clarity/testability/consistency.
4. Re-verify patched stories with fresh subagents.

# Output Contract

- corrected AC/schema items
- unresolved blockers/questions
- recommended implementation order if dependencies exist

# Done Criteria
- Story docs are testable and internally consistent.

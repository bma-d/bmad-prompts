# Agentic-JIT
MUST READ: @AGENTS.md, @.agents/agent-implementation-protocol.md, @.agents/jit-repo-overrides.md

# Role
Feedback implementation orchestrator.

# Goal
Implement latest feedback spec with strong verification and no scope drift.

# Inputs
- Latest file from @docs-feedback/
- Test guidance in @frontend/.agents/testing.md and @scripts/run-e2e-isolated.sh

# Procedure
1. Read latest feedback file and extract actionable items.
2. Launch planning subagents to map items -> files/tests.
3. Implement via parallel subagents with isolated ownership.
4. Run related tests; fix regressions.
5. Launch review subagents for changed areas.
6. Summarize results by feedback item.

# Output Contract
- Code/tests updated to satisfy acceptance criteria.
- Per-item status (`done|partial|blocked`) with evidence.
- Changelog updates for QA.

# Done Criteria
- No unresolved critical regressions.
- Related tests pass or blockers explicitly documented.

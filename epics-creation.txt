# Agentic-JIT
MUST READ: @AGENTS.md, @.agents/agentic-jit-framework.md, @.agents/agent-implementation-protocol.md

# Role
Epic/story generation orchestrator.

# Goal
Create or update epic/story docs from feedback with correct numbering and cross-links.

# Inputs
- Workflow: @_bmad/bmm/workflows/3-solutioning/create-epics-and-stories/workflow.md
- Source feedback file (provided by user or latest in @docs-feedback/)
- Existing epics index: @_bmad-output/planning-artifacts/epics/index.md

# Procedure
1. Resolve source feedback file and existing epic numbering.
2. Run create-epics-and-stories workflow.
3. Generate/update epic files in @_bmad-output/planning-artifacts/epics/.
4. Verify numbering continuity, links, and acceptance criteria completeness.
5. Update related index files.

# Output Contract
- New/updated epic + story docs are internally consistent.
- Numbering conflicts resolved.
- Summary of created/updated files.

# Done Criteria
- No missing links/sections.
- Epics trace back to source feedback items.

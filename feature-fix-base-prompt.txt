# Agentic-JIT
MUST READ: @AGENTS.md

# Role
Feature-fix execution base prompt.

# Goal
Fix user-reported issues by grounding in implementation artifacts and story context.

# Inputs
- User feedback/issues
- Relevant stories in @_bmad-output/implementation-artifacts/
- Planning docs in @_bmad-output/planning-artifacts/

# Procedure
1. Identify affected stories before coding.
2. Launch subagents to gather current behavior evidence.
3. Implement focused fixes with isolated ownership.
4. Run related tests and fix regressions.
5. Update impacted story progress/docs if required.
6. Update changelog entries for QA traceability.

# Rules
- Do not include unrelated work.
- Keep fixes surgical.
- Escalate if requirements conflict.

# Output Contract
- Summary of fixes, tests, and impacted stories.
- Explicit residual risks/blockers.

# Agentic-JIT
MUST READ: @AGENTS.md

# Role
Epic implementation review orchestrator.

# Goal
Verify that stories under target epics were implemented correctly and safely.

# Targets
- `_bmad-output/planning-artifacts/epics/epic-49-calendar-time-slot-visual-accuracy.md`
- `_bmad-output/planning-artifacts/epics/epic-50-appointment-visual-indicators-expansion.md`
- `_bmad-output/planning-artifacts/epics/epic-51-insurance-copay-alerts.md`
- `_bmad-output/planning-artifacts/epics/epic-52-bulk-message-time-filtering.md`
- `_bmad-output/planning-artifacts/epics/epic-53-appointment-activity-feed.md`
- `_bmad-output/planning-artifacts/epics/epic-54-patient-waitlist-system.md`
- `_bmad-output/planning-artifacts/epics/epic-55-patient-document-center.md`

# Procedure
1. Split stories across review subagents.
2. For each story, verify acceptance criteria vs code behavior + tests.
3. Consolidate findings with severity.
4. Dispatch fix subagents for validated issues.
5. Re-review changed areas.
6. Repeat until no unresolved `critical` findings.

# Output Contract
Comprehensive review summary with:
- per-story verdict (`pass|fail|blocked`)
- changed files/tests
- remaining risks

# Done Criteria
- No unresolved critical regressions.
- QA changelog entries updated.

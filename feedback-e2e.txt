# Agentic-JIT
MUST READ: @AGENTS.md

# Role
Feedback-to-E2E coverage verifier. Orchestrate subagents.

# Goal
Ensure latest feedback fixes are guarded by proper E2E regression tests.

# Inputs
- Latest feedback file from @docs-feedback/
- Existing E2E suites + isolated runner docs

# Procedure
1. Read latest feedback file and enumerate items.
2. For each item, verify fix implementation exists.
3. Decide E2E need:
- required
- optional
- unnecessary (with reason)
4. Add/update E2E tests for required items.
5. Run related E2E tests in isolation and fix issues.
6. Run reviewer subagents on new tests; autofix valid issues.

# Output Contract
Per-feedback-item E2E status:
- `covered-existing`
- `covered-new`
- `not-required`
- `blocked`

# Done Criteria
- No implemented feedback item lacking needed E2E coverage.
